## 项目列表

### schedule-card（日程管理卡片）

HarmonyOS NEXT 服务卡片（Form Widget）示例项目，使用 ArkTS + ArkUI 开发。

- **三种卡片尺寸**：2×2（小卡片）、2×4（中卡片）、4×4（大卡片）
- **信息展示**：日程标题、摘要、时间、优先级标签、完成进度
- **交互操作**：分类切换按钮（工作/生活/学习）、List 上下滑动浏览、点击跳转主应用
- **技术栈**：FormExtensionAbility、postCardAction、formBindingData、ArkUI 声明式 UI

使用 DevEco Studio NEXT 打开 `schedule-card/` 目录即可编译运行。
