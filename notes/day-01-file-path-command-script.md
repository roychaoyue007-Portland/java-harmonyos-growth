### 补充：为什么“工作目录不存在”会报错？

“工作目录不存在”的意思是：程序或命令想进入某个文件夹，但电脑找不到这个路径。

常见原因：

- 路径写错了。
- 文件夹被删除了。
- 盘符写错了，比如应该是 `E:`，却写成了 `D:`。
- 你以为文件夹在这里，但实际在别的位置。

例子：

```powershell
cd E:\Codex\abc
```

如果 `E:\Codex\abc` 这个文件夹不存在，PowerShell 就无法进入它，于是会报错。

所以遇到这个错误时，第一步应该检查路径是否真的存在。
