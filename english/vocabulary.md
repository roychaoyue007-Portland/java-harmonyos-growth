# 技术英文词汇表

## Day 01

| English | 中文 | 例子 |
| --- | --- | --- |
| repository | 仓库 | This repository contains my Java demos. |
| commit | 提交 | I made my first commit today. |
| branch | 分支 | Create a new branch for this feature. |
| push | 推送 | Push local changes to GitHub. |
| pull | 拉取 | Pull the latest code from GitHub. |
| clone | 克隆 | Clone the project to your computer. |
| class | 类 | Student is a Java class. |
| object | 对象 | Create a Student object. |
| method | 方法 | The study method prints a message. |
| field | 字段 | name is a field of Student. |

## Day 02

| English | 中文 | 例子 |
| --- | --- | --- |
| operating system | 操作系统 | Windows is an operating system. |
| CPU | 中央处理器 | The CPU executes program instructions. |
| memory | 内存 | Running programs use memory. |
| disk | 硬盘 / 磁盘 | The disk stores files and programs. |
| program | 程序 | A program is a set of instructions. |
| task manager | 任务管理器 | Open Task Manager to check CPU and memory usage. |
| process | 进程 | A running program usually has a process. |
| run | 运行 | Click the icon to run the program. |
| load | 加载 | The operating system loads the program into memory. |
| storage | 存储 | A disk is used for long-term storage. |
