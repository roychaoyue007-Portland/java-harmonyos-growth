# Day 01 - Git 和 GitHub 入门

## Git 是什么

Git 是一个版本管理工具。

它可以帮我记录代码每一次变化。如果代码改坏了，我可以找到以前的版本。

## GitHub 是什么

GitHub 是一个放代码的网站。

Git 是工具，GitHub 是平台。它们不是同一个东西。

## 常见词

- repository：仓库，用来保存一个项目
- commit：提交，保存一次代码变化
- branch：分支，用来同时开发不同功能
- push：把本地代码上传到远程仓库
- pull：把远程代码拉到本地
- clone：复制一个远程仓库到本地

## 常用命令

```bash
git init
git status
git add .
git commit -m "first commit"
git branch
git log
```

## 我今天要理解的一句话

Git 负责记录代码变化，GitHub 负责托管和协作。
