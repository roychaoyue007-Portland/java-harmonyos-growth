# Day 01 - 为什么我要重新学习计算机基础

日期：2026-06-14

## 当前情况

我是 Java / 鸿蒙程序员，但我发现自己还有很多基础不扎实的地方：

- 很多计算机基础概念只听过，但不能清楚解释
- Java 有些语法会用，但不知道底层原因
- GitHub、开源项目、技术论坛还不熟
- 英文技术资料阅读困难
- 学习容易零散，没有形成自己的知识系统

## 这 30 天的目标

我不追求 30 天学完所有知识，而是先建立稳定的学习节奏。

我希望 30 天后能做到：

- 每天能持续写代码和做笔记
- 能用自己的话解释一些基础概念
- 能看懂简单 Java 项目的目录结构
- 能开始阅读英文 README
- 能把学习内容沉淀到自己的仓库里
