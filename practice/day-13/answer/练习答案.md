# Day 13 练习答案：错误信息、退出码、日志与基础排错流程

日期：2026-06-29  
主题：错误信息、退出码、日志与基础排错流程  

## 一、三类安全错误练习

### 错误 1：路径不存在

执行命令：

```powershell
Get-ChildItem .\not-exist-folder
```

问题类型：

```text
路径不存在。
```

判断原因：

`Get-ChildItem` 这个命令本身存在，但传入的 `.
ot-exist-folder` 路径不存在。

验证方式：

```powershell
Get-Location
Test-Path .\not-exist-folder
Resolve-Path .
```

应该记住：

```text
路径不存在不是命令不存在，而是命令要操作的目标位置找不到。
```

### 错误 2：命令不存在

执行命令：

```powershell
not-a-real-command
```

问题类型：

```text
命令不存在。
```

判断原因：

PowerShell 无法把 `not-a-real-command` 识别为 Cmdlet、函数、脚本或外部程序。

验证方式：

```powershell
Get-Command not-a-real-command
```

应该记住：

```text
命令不存在是 Shell 找不到要执行的命令。
```

### 错误 3：Java 类不存在

执行命令：

```powershell
java NotExistClass
$LASTEXITCODE
```

问题类型：

```text
Java 运行时找不到指定类。
```

判断原因：

如果 `java` 命令能启动，说明 Shell 已经找到了 `java.exe`。失败点更可能是 Java 运行时找不到 `NotExistClass`，而不是 Java 没安装。

验证方式：

```powershell
Get-Command java
java -version
$LASTEXITCODE
```

应该记住：

```text
java 能启动，不等于 Java 要运行的类能找到。
```

## 二、今日排错流程

```text
1. 记录我执行的完整命令。
2. 保留完整错误信息。
3. 找出最重要的关键词：路径、命令名、类名、异常名、端口号。
4. 确认当前路径和环境。
5. 提出最可能的原因。
6. 用一条明确命令验证这个原因。
7. 如果不成立，再换下一个假设。
```

## 三、今日练习结论

路径不存在、命令不存在和 Java 类不存在是三类不同问题：

- 路径不存在：命令存在，但目标路径找不到。
- 命令不存在：Shell 找不到要执行的命令。
- Java 类不存在：`java` 命令能运行，但 Java 运行时找不到指定类。
