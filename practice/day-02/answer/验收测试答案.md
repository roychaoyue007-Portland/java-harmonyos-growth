# Day 02 验收测试答案

> 来源：`验收测试答案.docx` 转换为 Markdown。

8 道题答案：

1.windows是一个操作系统，负责协调CPU、内存等硬件，使他们有序执行各自的工作。

2.CPU主要负责实际干活，比如运行程序等。

3. 因为内存用来临时保存正在运行的程序和数据。程序关闭或电脑关机后，这些数据通常会被释放，不像硬盘那样长期保存。

4. 因为硬盘是长期存储设备，断电后数据仍然可以保留，所以文件、软件、图片、代码都可以长期放在硬盘里。

5.硬盘

6.内存

7.程序打开，从硬盘加载到内存，CPU再执行

8. CPU 表示当前电脑的计算负载；内存表示正在运行的程序占用了多少临时空间；磁盘表示硬盘正在进行多少读取和写入操作。
