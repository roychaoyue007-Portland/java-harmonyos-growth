### 2. 文件、文件夹、路径分别是什么？

原回答：

> 文件是存储数据的最小单位，如图片，word文件等；文件夹是存放文件或者其他文件夹的地方；路径是这个文件或者文件夹的地址。

得分：10 / 10。

整理：文件是存储数据的基本单位；文件夹是存放文件或其他文件夹的位置；路径是文件或文件夹的位置描述。

### 3. 什么是绝对路径？请举一个 Windows 例子。

原回答：

> 从根目录开始的路径；C：\codex

得分：8 / 10。

整理：绝对路径是从根位置开始写出的完整路径。

正确示例：

```text
C:\Codex
```

注意：Windows 盘符后面使用英文冒号 `:`。

### 4. 什么是相对路径？它相对于什么？

原回答：

> 从当前位置开始的路径；相对于当前目录。

得分：10 / 10。

整理：相对路径是从当前目录开始描述目标位置的路径。

### 5. 当前目录变化后，为什么同一条相对路径可能指向不同位置？

原回答：

> 因为相对路径就是基于的当前目录。

得分：8 / 10。

整理：因为相对路径的起点是当前目录。当前目录改变，相对路径的起点也改变，所以同一条相对路径可能指向不同位置。

### 6. `.` 和 `..` 分别表示什么？

原回答：

> 表示当前目录；表示当前目录的父目录。

得分：10 / 10。

整理：

```text
. 表示当前目录。
.. 表示当前目录的父目录。
```

### 7. 请解释 `.\notes\today.txt` 这条路径的含义。

原回答：

> notes目录下的today.txt文件。

得分：8 / 10。

整理：

```text
从当前目录出发，进入 notes 文件夹，找到 today.txt 文件。
```

### 8. 假设当前目录是 `C:\study\day09-path-demo`，那么 `..\day08` 大致指向哪里？

原回答：

> 大致指向study，day08和day09应该是同级目录，..代表父目录，那么../day08就是指向day09的父目录，也就是study。

得分：6 / 10。

整理：

```text
当前目录：
C:\study\day09-path-demo

.. 回到父目录：
C:\study

再进入 day08：
C:\study\day08
```

最终指向：

```text
C:\study\day08
```

关键提醒：

```text
.. 不是最终目的地。
.. 只是先回到父目录。
后面的 day08 还要继续进入。
```


### 补充验收点：`Resolve-Path`

正确理解：

```text
Resolve-Path 用来把相对路径解析成完整路径，并确认路径真实存在。
```

示例：

```powershell
Resolve-Path .
Resolve-Path ..
```

整理：

```text
Resolve-Path .  可以看到当前目录的完整路径。
Resolve-Path .. 可以看到父目录的完整路径。
```

### 补充验收点：路径存在 vs 路径可访问

正确理解：

```text
路径存在：文件或文件夹真实存在。
路径可访问：当前用户或程序有权限访问它。
```

两者不同。一个路径可能存在，但因为权限原因不能访问或不能修改。
## 三、最终知识整理

### 文件系统

```text
文件系统负责组织、保存、查找和管理文件。
```

### 文件、文件夹、路径

```text
文件：保存具体数据。
文件夹 / 目录：存放文件或其他文件夹。
路径：文件或文件夹的位置描述。
```

### 绝对路径与相对路径

```text
绝对路径：从根位置开始，不依赖当前目录。
相对路径：从当前目录开始，依赖当前目录。
```

### `.` 与 `..`

```text
.  表示当前目录
.. 表示父目录
```

### 相对路径推理

看到：

```text
..\xxx
```

就按两步想：

```text
第一步：.. 回到父目录
第二步：进入 xxx
```
