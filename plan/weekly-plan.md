# Weekly Plan

## 执行规则

这是当前每天学习安排的唯一执行计划。

当我说“继续全栈学习计划，今天是 Day XX”时，请按本文件里的 Day XX 执行。

不要使用 `archive/old-30-day-plan/` 里的旧 30 天 Java 计划，除非我明确要求查看旧计划。

当前周目标：理解计算机世界最基础的语言。

本周不要急着学 Java、Spring Boot、Vue 或鸿蒙框架。先把文件、路径、命令、脚本、操作系统、程序运行这些概念弄清楚。

## Day 01 - 文件、文件夹、路径、命令、脚本

建议学习时长：3 到 4 小时。

### 1. 概念理解：60 分钟

理解这 5 个词：

- 文件
- 文件夹 / 目录
- 路径
- 命令
- 脚本

每个词都要能回答：

- 它是什么？
- 它解决什么问题？
- 我在哪里会遇到？
- 它和哪些词有关？
- 我有没有亲手做过？

### 2. PowerShell 操作：60 分钟

练习命令：

```powershell
pwd
dir
cd E:\
dir
cd E:\Codex
dir
cd E:\Codex\projects\java-harmonyos-growth
dir
mkdir practice\day-01\my-first-folder
dir practice\day-01
```

### 3. 今日笔记：40 分钟

笔记文件：

- `notes/day-01-file-path-command-script.md`

需要回答：

- 文件是什么？
- 文件夹是什么？
- 路径是什么？
- 当前目录是什么？
- `cd` 是什么？
- `dir` 是什么？
- 脚本是什么？

### 4. 技术词典：30 分钟

更新文件：

- `dictionary/tech-dictionary.md`

记录 5 个词：

- 文件
- 文件夹
- 路径
- 命令
- 脚本

### 5. 验收测试：30 到 40 分钟

回答 8 道题：

1. `E:\Codex\projects\java-harmonyos-growth` 这条路径表示什么意思？
2. PowerShell 里显示 `PS E:\Codex>`，这代表什么？
3. 如果在 `E:\Codex` 下执行 `dir`，会看到什么？
4. `cd E:\Codex\projects` 是什么意思？
5. `mkdir test-folder` 做了什么？
6. 为什么“工作目录不存在”会报错？
7. 脚本和命令有什么区别？
8. 如果一个脚本里有 5 行命令，电脑会怎么执行？

### 6. 复盘：10 到 20 分钟

写下：

- 今天我学会了什么？
- 今天我还不懂什么？
- 今天最重要的一个概念是什么？
- 明天我想继续补什么？
