# Six Month Fullstack Plan

目标：用 6 个月从基础不稳的 Java / 鸿蒙开发者，成长为能独立完成一个全栈项目的开发者。

核心原则：先听懂软件开发世界的语言，再进入具体技术栈。

## 四条主线

1. 概念线：理解术语、原理和它们之间的关系。
2. 操作线：会使用电脑、命令行、IDE、Git、调试和部署工具。
3. 编程线：能用 Java / JavaScript / ArkTS 写出真实功能。
4. 项目线：最终完成 Java 后端 + 数据库 + 鸿蒙客户端的完整项目。

## Month 01 - 计算机导论与操作系统入门

目标：听懂最基础的软件开发语言。

主题：

- 文件、文件夹、路径、目录
- 终端、命令、Shell、脚本
- 程序、进程、线程
- CPU、内存、硬盘、操作系统
- 文件系统、环境变量、安装、配置
- 编译、运行、解释执行

产出：

- 技术词典 50 到 80 个词
- 命令行练习记录
- 计算机基础笔记

## Month 02 - 编程基本功与 Java 核心

目标：理解程序如何表达逻辑。

主题：

- 变量、类型、判断、循环
- 函数、方法、类、对象
- 数组、字符串、异常、输入输出
- 封装、继承、多态
- 接口、抽象类、泛型、集合
- Debug 和报错阅读

产出：

- 学生管理系统
- 图书管理系统
- 控制台 Todo
- Java 基础笔记

## Month 03 - 数据结构、数据库与接口基础

目标：进入后端开发前的核心地基。

主题：

- 数组、链表、栈、队列、哈希表
- 排序、查找、递归
- MySQL、表、字段、主键、索引
- SQL 增删改查
- HTTP、JSON、接口、请求、响应

产出：

- 通讯录小程序
- MySQL 练习库
- 接口概念笔记

## Month 04 - Spring Boot 后端

目标：能做一个可用的 Java 后端。

主题：

- Spring Boot
- Controller、Service、Mapper / Repository
- REST API、Maven / Gradle、依赖
- 配置文件、参数校验、异常处理
- 登录注册、Token / JWT、日志

产出：

- todo-backend
- 用户注册登录
- 任务管理接口
- 接口文档

## Month 05 - 客户端、鸿蒙与工程化

目标：理解用户界面如何调用后端。

主题：

- 页面、组件、状态、路由
- 表单、列表、事件、网络请求
- ArkTS、ArkUI、HarmonyOS 项目结构
- HTML / CSS / JavaScript 基础认识
- Node.js、npm、Vite 基本认识
- Git、GitHub、README、分支、提交、合并

产出：

- todo-harmonyos
- 可选 todo-web
- GitHub 项目仓库

## Month 06 - 部署与完整全栈项目

目标：做出一个能展示的完整作品。

主题：

- Linux、服务器、端口、Nginx
- 部署、打包、环境变量、日志排查
- 开发环境 / 生产环境
- 项目 README、截图、演示说明

最终项目：

```text
fullstack-final-project
  backend-springboot
  harmonyos-client
  docs
  README.md
```

可选增加：

```text
web-vue
```

## 每天 3 到 4 小时模板

1. 45 分钟：读书或概念理解。
2. 45 分钟：拆技术词，写技术词典。
3. 90 分钟：动手练习或写代码。
4. 30 分钟：复盘笔记。
5. 可选 30 分钟：英文 README、报错或文档阅读。

## 每天必须产出

- 3 到 5 个技术词解释
- 1 个操作练习或代码练习
- 1 篇简短复盘
